# Operational Loop

This document captures the public-safe operating loop implied by the current simulator baseline.

1. Start or continue a manager run.
2. Review storeroom inventory and fill gaps.
3. Open vendor workspace and inspect catalog options.
4. Draft and review current orders.
5. Receive or stage inventory for route use.
6. Load machines by slot and depth.
7. Review reports for the current operational state.

The value of the simulator is that these steps are connected. Inventory decisions affect orders, orders affect loadout options, and all of that rolls up into reporting surfaces.
