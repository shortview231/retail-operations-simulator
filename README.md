# Retail Operations Simulator

Public-facing baseline repository for Luna's retail simulation work.

The current internal evidence points to two real foundations:

- an earlier SQL + Python POS simulator portfolio project
- an active vending-first retail simulator project with SQLite schema and operations-oriented documentation

This first public baseline is intentionally modest. It documents the simulator direction truthfully instead of pretending the public repo already contains the entire internal build.

## What exists today

- A simple SQL + Python POS simulator with SQLite and Tkinter
- An active vending-first retail simulator project in Lucid Vision
- A documented Phase 1 schema registry for the vending simulator
- Existing setup and run scripts for the simulator database and run loop
